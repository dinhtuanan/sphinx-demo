.. _open:

Opening a new subtitle file
---------------------------

To create and new subtitle:
1. Click on the "File" menu.
2. Click on **New** |newbutton| button

.. |newbutton| image:: images/fish.jpg
               :scale: 5 %
.. _import:

Import Subtitles from an external file
--------------------------------------

To import subtitles from an external file:

.. image:: images/fish.jpg

1. Click on the "File" menu.
2. Click on **Import** button
